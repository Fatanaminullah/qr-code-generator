# QR Code Generator

Simple website to generate a QR code from a URL (no ads, no expiry). The QR is generated locally in your browser.

## Features

- Enter a URL
- Click **Generate**
- See the QR image
- Click the QR (or the **Download PNG** button) to download

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```
